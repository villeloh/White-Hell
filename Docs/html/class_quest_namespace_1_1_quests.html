◆ ChangeQuestIcon()

void QuestNamespace.Quests.ChangeQuestIcon ( GameObject  givenIcon)

Changes the quest icon to one that indicates that the quest has been completed.

The collider is destroyed in order to make the quest non-interactable. If the icon is tagged as a shelter, then the method changes its icon to the 'shelter' icon, but leaves the collider intact.
